Query/Command library contains some examples for using the first tab of TeklaPad.

It's available for both Model & Drawing spaces.

 

 

Hint: The main benefit of using first tab of TeklaPad (instead of second tab) is that the first tab will works in all versions of Tekla Structures without need to any additional configuration. In addition, this tab has an exclusive xAPI to simplify communicating with Tekla Database. Find more on:

User Manual > Tab-1 > C# basics > Extended API

 

 

 

You can also share your elegant queries/commands with us to improve the library.
