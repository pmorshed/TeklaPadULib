You can use second tab of TeklaPad to create plug-ins with inheriting PluginBase class.

 

 

 

 

 

 

 

 

 

 

Step by step procedure:

1- Run the project.

2- Rename the created *.dll file (*.dll name must be same as plug-in name in your code and cannot contains spaces).

3- Create new folder in ...\TeklaStructures\[version]\nt\bin\plugins. (folder name must be same as plug-in name).

4- Move (or copy) the renamed *.dll file to created folder. This adds your plug-in directly to the "Applications & components" catalog in Tekla Structures.

(Note: *.dll name and plug-in name and folder name must be same).

3- Run (or restart) the Tekla Structures and look for the plug-in in the "Applications & components" catalog.

 

Important:

All created *.dll files will be removed automatically after closing the TeklaPad.

So, you should move/copy your final plug-in to another location (for later usages) before closing the TeklaPad.
