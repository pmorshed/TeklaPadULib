This section contains some examples for drawing commands.

 

Note1: The big benefit of using first tab of TeklaPad (instead of second tab) for creating commands is that the first tab will work in all versions of Tekla Structures without need to any additional configuration and also you can use the extended features (like DrawingDatabase, GetTeklaReport(),...) to simplifying the procedure. Find more about extended features (xAPI) on: User Manual > Tab-1 > C# basics > Extended API
